Este é um documento HTML simples criado para testar a conversão para PDF.

Instruções de Uso

Para converter este HTML em PDF, você pode fazer o seguinte:

  1. Copie todo este código HTML
  2. Faça uma requisição POST para /convert incluindo o HTML no corpo da requisição

Exemplo de curl

curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"htmlContent":"<html><body><h1>Teste</h1></body></html>"}' \
  https://bmo-relatorios.vercel.app/convert \
  --output documento.pdf

Recursos adicionais

  • O conversor suporta todos os recursos de HTML moderno
  • Você pode incluir imagens e estilos CSS
  • Tabelas e formatação complexa são suportadas
